- GitToolBox 每行代码显示git作者
- Extra Icons 图标更直观
- Background Image Plus + 背景图设置
- .env files  env支持工具
- .ignore 高亮GIT忽略文件
- nginx support 运行/重新加载配置/停止nginx服务器的想法。配置文件语法突出显示，自动完成和检查
- Swoole IDE Helper swoole帮助文件
- BashSupport Pro 可以书写.sh脚本，且同样可以在书写时有对应的语法提示功能
- ideolog 查看日志
- Rainbow Brackets 彩虹括号
- Grep Console 方便查看日志, 对不同日志的字体标记颜色
- SonarLint 编码不规范时，它就如站在你身边的大牛，提醒你
- CodeGlance 代码地图插件 , 类似SublimeText的Mini Map插件
- CamelCase 命名风格转换插件
- Git Commit Template Git格式化模版
- Key Promoter X 督促你使用快捷键
- Jms messenger 查看、修改消息队列的插件
- PHP Annotations PHP 注解
- Elasticsearch 更好用的ES界面

- Elasticsearch Query - EDQL 像文本一样记录和执行ES语句
- Translate 翻译插件
- Material Theme UI UI主题插件
- Atom Material Icons UI主题图标
- Laravel Idea laravel 语法支持插件,在使用view route config函数时，提示对应的所有路径和值的自动完成功能
- Laravel Make Integration 快速创建controller、model
- Laravel Tinker 方法参数提示
- Laravel Live Templates 常用代码语句块
- Laravel Query 查询语句提示，不用输query语句就能提示
- Laravel Generator
```
将数据库模式转换为Laravel迁移  
运行工匠命令  
设计数据库表  
定义模型的工厂  
管理模型的关系  
自定义层(控制器、模型、存储库等)  
自定义模板(语法\:for, if等)  
生成控制器，迁移，模型，测试等。
```
