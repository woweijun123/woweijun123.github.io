# 配置目录
```shell
# windows
%APPDATA%\JetBrains\<产品><版本>
# macos
~/Library/Application Support/JetBrains/<产品><版本>
# linux
~/.config/JetBrains/<产品><版本>
```
# 系统目录
```shell
# windows
%LOCALAPPDATA%\JetBrains\<产品><版本>
# macos
~/Library/Caches/JetBrains/<产品><版本>
# linux
~/.cache/JetBrains/<产品><版本>
```
# 插件目录
```shell
# windows
%APPDATA%\JetBrains\<产品><版本>\plugins
# macos
~/Library/Application Support/JetBrains/<产品><版本>/plugins
# linux
~/.local/share/JetBrains/<产品><版本>
```
# 日志目录
```shell
# windows
%LOCALAPPDATA%\JetBrains\<产品><版本>\log
# macos
~/Library/Logs/JetBrains/<产品><版本>
# linux
~/.cache/JetBrains/<产品><版本>/log
```
